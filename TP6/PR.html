Abstract:

Para este trabalho prático foi criado um servidor web (index.js) para gerir terafas. Este gestor de tarefas permite ao utilizador adicionar uma tarefa, indica a sua descrição e a data em que a tarefa até que a tarefa seja finalizada, ainda foi adicionada a funcionalidade de remover tarefas

O armazenamento das tarefas é feita num ficheiro JSON, tarefas.json

Para gerar o HTML desejado foi utilizada o pug, isto é, os ficheiros layout.pug (layout dos seguintes ficheiros), erro.pug (geração de páginas de erro) e index.pug (o gestor de tarefas)

Ainda foi feito, como trabalho adicional, a remoção das tarefas pelo utilizador, neste caso, embora não seja a forma convencionada, é feita através de um método post, visto que não foi encontrada um alternativa plausível , na minha opinião, aos forms que não suportam o método delete. Mesmo assim, esta funcionalidade também é tratada pelo servidor caso haja um método delete válido, verificada pela ferramenta Postman, pois pela aplicação em si não o é possível com a versão atual
