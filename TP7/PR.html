Abstract:

Para este trabalho prático foi criado um servidor web que permite utilizar as operações CRUD num base de dados em mongoDB

Esta aplicação foi desenvolvida com o modelo arquitetural MVC

Para gerar o HTML desejado foi utilizada o pug, isto é, os ficheiros layout.pug (layout dos seguintes ficheiros), filme.pug (geração da página de um filme) erro.pug (geração de páginas de erro), form.pug (edição de filmes) e index.pug (página inicial da aplicação)

Na página inicial encontra-se a lista dos filmes, ordenados alfabéticamente, indicando o titulo e o ano de estreia, caso o utilizador queira visualizar um filme deve clicar no título do filme, caso queira adicionar um novo filme deve clicar no butão no topo da página, caso queira editar ou eliminar deve clicar no botão associado ao filme que se localiza à direita deste. A edição do filme também pode ser feita na culta do respetivo filme

Para não haver o carregamento de todos os filmes na página inicial, o que torna o carregamento da página lento por haver o transporte de uma grande quantidade de dados foi adotado um sistema de página, em que apenas são carregados no máximo 30 filmes. Para mudar de página o utilizador deve dirigir-se ao canto superior direito da página.

Em qualquer página que não seja a inicial, é possivel voltar a esta ao clicar no link que se encontra no fim da página
