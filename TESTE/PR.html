Abstract:

De seguida irei explicar como inserir os dados na base de dados MongoDB e como executar o servidor

Assume-se que esteja numa linha de comandos ou numa consola, primeiro deve mudar-se para a pasta ex1 (cd ex1), segundo deve instalar todos os modulos (npm i), terceiro deve inserir os dados no ficheiro JSON na base de dados (mongoimport -d musicas -c musicas --jsonArray arquivo-musical.js), quarto deve executar o servidor (npm start)

O ficheiro JSON foi convertido do ficheiro XML utilizando o site convertjson.com e removendo o campo obra como raiz

Para a query que envolve um id foi escolhido o id fornecido pelo ficheiro JSON e não o gerado pelo MongoDB.

Quanto ao exercício 2, o ficheiro com as respostas às perguntas de aquecimento é o Respostas.txt

Para iniciar o servidor de interface deve-se dirigir-se à pasta ex2 e executar o comando npm start

Qualquer um destes servidores correram na porta por defeito, ou seja 3000, caso queira aceder à interface Web do exercicio 2 deve aceder o link http://localhost:3000/ , caso seja para testar os dados do exercício 1 deve usar este path.

O servidor de MongoDB é usado na sua porta por defeito
